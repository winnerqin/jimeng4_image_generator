# 如何查看浏览器控制台输出

## 方法一：使用快捷键（推荐）

### Chrome/Edge 浏览器
- **Windows/Linux**: 按 `F12` 或 `Ctrl + Shift + I`
- **Mac**: 按 `Cmd + Option + I`

### Firefox 浏览器
- **Windows/Linux**: 按 `F12` 或 `Ctrl + Shift + I`
- **Mac**: 按 `Cmd + Option + I`

### Safari 浏览器
- 首先需要启用开发者菜单：
  1. 打开 Safari 偏好设置
  2. 在"高级"标签中勾选"在菜单栏中显示开发菜单"
- 然后按 `Cmd + Option + C`

## 方法二：通过菜单

### Chrome/Edge
1. 点击右上角三个点（菜单）
2. 选择"更多工具" → "开发者工具"

### Firefox
1. 点击右上角三条横线（菜单）
2. 选择"更多工具" → "Web 开发者工具"

## 查看控制台日志

打开开发者工具后：
1. 点击"Console"（控制台）标签
2. 你会看到类似这样的日志：
   ```
   [轮询 14:30:25] 任务 abc123 状态: generating
   [轮询 14:30:26] 任务 abc123 状态: generating
   [轮询 14:30:27] 任务 abc123 状态: completed
   ✅ 任务完成！
   ```

## 调试信息

页面上也会显示一个调试信息框（右上角），显示：
- 当前时间
- 任务状态
- 任务ID

## 常见问题

### 如果看不到日志
1. 确保控制台已打开
2. 检查是否有过滤器（清除过滤器）
3. 刷新页面重新开始

### 如果任务一直显示"生成中"
查看控制台日志，检查：
- 任务状态是什么（generating/completed/failed）
- 是否有错误信息
- 轮询是否在正常工作

## 清除控制台

在控制台中点击清除按钮（🚫）或按 `Ctrl + L`（Mac: `Cmd + K`）
